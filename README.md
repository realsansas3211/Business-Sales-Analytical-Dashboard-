# Business-Sales-Analytical-Dashboard
A professional, end-to-end Power BI sales performance dashboard designed for monitoring revenue, transaction health, and business growth.

Project Overview

This dashboard provides a complete view of business sales operations with a focus on:
-Total Transaction Value: $24.961 Trillion
- Total Net Amount: $24.846 Trillion
- Total Transactions: 10,000+

It delivers real-time insights into revenue trends, customer behavior, and operational efficiency.

Key Features

-Executive KPI Cards: Total Amount, Net Amount, and Transaction Count
-Transaction Health: Success, processing, failed, and pending status analysis
-Card Performance: Revenue by payment card type
-Geographic Analysis: Total net amount by city
-Temporal Trends: Monthly revenue trend with seasonality
- Behavioral Insights: Weekday vs Weekend transaction comparison

Key Insights Delivered

- Monthly revenue patterns and peak seasons
- Transaction success rate and operational bottlenecks
- High-performing cities and geographic opportunities
- Payment method preferences
- Weekday vs Weekend customer behavior

Tech Stack

- Power BI Desktop
- DAX for complex calculations and KPIs
- Power Query for data transformation
- Modern Dashboard Design principles

How to Use

1. Download or clone the repository
2. Open `Business_Sales_Analytics.pbix` with Power BI Desktop
3. Interact with filters and drill into specific months, cities, or card types
4. Use the visuals to support business reporting and decision-making
